# Netflix_PowerBI_Project
This is a Power BI dashboard built to analyze Netflixâ€™s content. The dashboard provides insights into content distribution, genres, countries, release trends, and content types (Movies vs TV Shows). The goal of this project is to demonstrate data cleaning, transformation, DAX calculations, and data visualization skills using Power BI.

# Tools & Technologies Used
- Power BI
- Power Query (Data Transformation)
- DAX (Measures & Calculated Columns)
- CSV Dataset

# Dataset
netflix_titles.csv
Source: Publicly available Netflix titles dataset from Kaggle
Information includes Show_id, type, title, Director, Cast, Country, Date_added,release year, rating, duration, genre and description.

# Data Cleaning & Transformation
Performed using POWER QUERY:
- Removed null values
- Split multi-value columns (Genre, Country)
- Standardized column names
- Extracted primary genre and primary country
- Created new derived columns for analysis

# Dashboard Features
Key KPIs:
- Total Titles
- Total Movies
- Total TV Shows
- Distinct Genres
- Content-producing Countries

# Visuals Include:
- Movies vs TV Shows distribution
- Top genres by content count
- Top countries by number of titles
- Content addition trend by year
- Interactive slicers for Type, Genre, Country, and Release Year
